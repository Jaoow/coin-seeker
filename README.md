# COIN SEEKER GAME

Jogo "COIN SEEKER" criado em Javascript puro

## Imagens

![Game Preview](src/previews/preview_1.png)

## COMO JOGAR?

O objetivo do jogo é coletar todos as moedas na fase, até completar todas as fases e concluir o jogo

Controles: o jogo é possivel jogar com as **setinhas do teclado** ou o **WASD**
